## Что сделано

Реализован помехоустойчивый архиватор файлов без сжатия **HamArc**. Для помехоустойчивого кодирования было использовано [кодирование Хэмминга](https://en.wikipedia.org/wiki/Hamming_code).

## Возможности

* Объединение файлов в архив в формате .haf (Hamming Archive File)
* Извлечение файлов из архива
* Объединение нескольких архивов в один
* Восстановление архива при повреждениях
* Возврат списка файлов в архиве

## Реализация

Консольное приложение, поддерживающее следующие аргументы командной строки:

**-c, --create**           - Создание нового архива

**-f, --file=[ARHCNAME]**  - Имя файла с архивом

**-l, --list**             - Вывод списка файлов в архиве

**-x, --extract**          - Извлечение файлов из архива (если не указано, то все файлы)

**-a, --append**           - Добавление файла в архив

**-d, --delete**           - Удаление файла из архива

**-A, --concatenate**      - Объединение двух архивов

**-b, --bits**             - Установка количества бит для кодирования

**Имена файлов передаются свободными аргументами**

### Примеры запуска

*hamarc --create --file=ARCHIVE FILE1 FILE2 FILE3*

*hamarc -l -f ARCHIVE*

*hamarc --concantenate  ARCHIVE1 ARCHIVE2 -f ARCHIVE3*

### Примечание
    - Работает только на Windows
